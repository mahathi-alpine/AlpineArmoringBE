### `Initial setup`
1. **Git clone** the project
2. Run **yarn install**
3. Set [Environment Variables](https://www.postgresql.org/download/) (Copy/paste the whole file and just replace the DATABASE_PASSWORD)
4. Install [PostgreSQL](https://www.postgresql.org/download/)

### `develop`

Start the Strapi application

```
yarn run dev

```
